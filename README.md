# JavaScript Bot สำหรับการดึงข้อมูล Email - File แนบ ใน Outlook
## ติดตั้ง
I) ติดตั้ง puppeteer => `npm install puppeteer`

## การใช้งาน
ดาวน์โหลดไฟล์ และ ใช้ nodejs หรือ node ในการรันโปรแกรม `nodejs bot.js`
